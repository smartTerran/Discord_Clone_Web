# Discord_clone
A Discord clone made with Django. You can use it at [Discord clone].

## Getting Started

* Clone the repo: 

```bash
$ git clone [github repository link]

```

* Install dependencies: 

```bash
$ pip install -r requirements.txt

```

* Migrate: 

```bash
$ python manage.py migrate

```

* Run

```bash
$ python manage.py runserver

```
